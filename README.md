## ğŸš€ Getting Started

To run this project locally, follow these steps:

### Prerequisites

* Node.js (v16+)
* npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/VishruthC/musicapp.git](https://github.com/VishruthC/musicapp.git)

    ```

    ```bash
    npm install
    ```

